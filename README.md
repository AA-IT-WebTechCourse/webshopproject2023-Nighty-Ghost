## Install requirements.txt
(virtualenv initially activated) In `backend` folder or root folder, 
```bash
pip install -r requirements.txt
```

## Install NPM Modules
(virtualenv initially activated) In `frontend` folder

```sh
npm install
```

## RUN PROJECT
In backend folder:
```bash
python manage.py runserver    
```
In frontend folder:
```bash
npm start  
```
